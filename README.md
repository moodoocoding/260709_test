# 📝 교사용 오프라인 시험 채점 시스템

각 학급의 담임 교사가 개별적으로 사용할 수 있도록 최적화된 초간편 웹 기반 자동 채점 시스템입니다.

## ✨ 주요 특징

1. **완전한 오프라인 환경 지원 (No CORS)**:
   - 외부 서버와의 통신이나 API 호출이 전혀 없고, 정답 데이터가 자바스크립트 코드 내에 직접 내장(임베딩)되어 있습니다.
   - 외부 라이브러리(SheetJS, Chart.js)도 `lib/` 폴더에 로컬 저장되어 있어 **인터넷이 차단된 PC**에서도 `index.html` 파일을 더블클릭하는 것만으로 100% 동작합니다.

2. **담임교사 1인 학급 최적화**:
   - 담당하는 단일 학급(반 번호)과 우리 반 학생 수만 한 번 저장해두면 모든 탭의 학생 리스트와 성적표 형식이 자동 세팅됩니다.

3. **고속 키보드 채점 및 학생별 전환 오버레이**:
   - **객관식 문항**: 키보드 `1` ~ `5` 숫자 키로 즉시 입력
   - **주관식/서술형**: 화면의 정답 텍스트와 비교하여 키보드 `O` / `X` 입력
   - **학생 전환 자동 정지**: 한 학생의 채점(25문항)이 끝나는 즉시 총 점수와 맞춘 개수를 알려주는 팝업이 뜨며, `Enter`를 치면 다음 학생으로 빠르게 이행합니다.

4. **강력한 결과 분석 및 엑셀 다운로드**:
   - 학급 평균 점수, 문항별 정답률 통계, 점수 구간별 학생 수 분포를 Chart.js 그래프로 확인 가능합니다.
   - **엑셀 다운로드** 클릭 시 석차가 포함된 전체 성적표 및 문항분석표를 포함한 엑셀 보고서 파일이 다운로드됩니다.

---

## 🚀 실행 및 사용 방법

1. 본 폴더 내의 `index.html` 파일을 크롬(Chrome)이나 엣지(Edge) 등 웹 브라우저로 엽니다.
2. 첫 화면인 **시험 설정**에서 담임 반 번호와 학생 수를 입력하고 **설정 저장**을 누릅니다.
3. **정답 확인** 탭에서 내장된 정답이 정상적으로 나타나는지 확인합니다.
4. **답안 입력** 탭에서 과목을 선택하고 **채점 시작**을 누른 뒤 키보드로 채점을 진행합니다.
5. 완료 후 **성적 분석** 탭에서 결과를 검토하고 **엑셀 다운로드**를 수행하여 보관합니다.

---

## ⌨️ 채점 화면 키축 단축키

* `1` ~ `5` : 객관식 문항 정답 입력 (자동 다음 문제 이동)
* `O` / `X` : 주관식/복수정답 정오 판정 입력 (자동 다음 문제 이동)
* `←` (방향키 왼쪽) : 이전 문제로 이동
* `→` (방향키 오른쪽) : 다음 문제로 이동 (건너뛰기)
* `ESC` : 채점 중단 및 선택 화면으로 복귀
* `Enter` / `Space` : 오버레이창(시작/점수 확인 등) 닫고 채점 진행
